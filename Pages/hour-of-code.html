About hour of code event

The Hour of Code event was an inspiring experience where we introduced students to the basics of coding using Code.org. Despite challenges like limited devices and connectivity issues, we ensured every student could participate. They learned to create simple algorithms, use loops, and solve problems through interactive tasks. By the end, each student completed 20 activities and received a certificate, leaving them motivated and excited about coding and technology. The event not only taught new skills but also sparked curiosity for further learning.
